# Sistema de Solicitudes – Power Platform

Aplicación desarrollada con Power Apps y Power Automate.

## Funcionalidades
- Creación de solicitudes
- Aprobación / Rechazo
- Roles de usuario y administrador

## Tecnologías
- Power Apps (Canvas App)
- Power Automate
- SharePoint Online

## Instalación
1. Importar la Power App desde el paquete ZIP
2. Importar el flujo Power Automate
3. Crear lista SharePoint según documentación
